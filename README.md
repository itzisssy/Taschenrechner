# 🧮 Python Taschenrechner Projekt (Basic + Advanced)

Dieses Projekt enthält zwei Versionen eines Taschenrechners in Python.

---

# 📌 BASIC VERSION

Ein einfacher Taschenrechner für zwei Zahlen und eine Grundoperation.

## ➗ Operationen

- + Addition  
- - Subtraktion  
- * Multiplikation  
- / Division  

## 💡 Beispiel

Geben Sie die erste Zahl ein: 10  
Geben Sie die zweite Zahl ein: 5  
Wählen Sie die Operation (+, -, *, /): +  
15  

## ⚠️ Hinweise

- Nur ganze Zahlen
- Keine erweiterte Fehlerprüfung

---

# 📌 ADVANCED VERSION

Ein erweiterter Taschenrechner mit Fehlerprüfung und mehr Funktionen.

## ➗ Operationen

- + Addition  
- - Subtraktion  
- * Multiplikation  
- / Division (mit Schutz vor 0)  
- ^ Potenz  
- % Modulo  

## 🧠 Features

- Fehlerprüfung bei Eingaben  
- Ungültige Eingaben werden abgefangen  
- Endlosschleife (mehrere Rechnungen möglich)  
- Beenden mit "exit"  

## 💡 Beispiel

Geben Sie die erste Zahl ein (oder 'exit' zum Beenden): 10  
Geben Sie die zweite Zahl ein (oder 'exit' zum Beenden): 3  
Wählen Sie die Operation (+, -, *, /, ^, %) oder 'exit': ^  
Ergebnis: 1000.0

# 🧮 Python Taschenrechner (Basic & Advanced)

Ein einfaches Python-Projekt mit zwei Taschenrechner-Versionen:

- 🔹 Basic Version (einfach)
- 🔹 Advanced Version (erweitert mit mehr Funktionen & Fehlerbehandlung)

---

# 📌 1. Basic Taschenrechner

## ▶️ Beschreibung

Die Basic-Version ist ein einfacher Rechner für zwei Zahlen und eine Grundoperation.

---

## ➗ Unterstützte Operationen

- `+` Addition  
- `-` Subtraktion  
- `*` Multiplikation  
- `/` Division  

---

```text
Geben Sie die erste Zahl ein: 10  
Geben Sie die zweite Zahl ein: 5
